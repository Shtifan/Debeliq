# Instalation

Run:

```bash
git clone https://github.com/Shtifan/Debeliq.git
npm i
```

# Config

Make config.json file:

```json
{
    "clientId": "",
    "token": "",
    "YT_credentials": ""
}
```

For YouTube credentials run:

```bash
npx --no discord-player-youtubei
```
